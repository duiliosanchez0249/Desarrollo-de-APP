# Gaming Hub API Pro

Bienvenido a **Gaming Hub API**, un sistema profesional para la gestión de videojuegos y usuarios, desarrollado con **FastAPI** y **SQLModel**.

## Descripción del Dominio
Este proyecto permite gestionar un catálogo de videojuegos, permitiendo el registro de títulos, géneros y calificaciones. Incluye un sistema de gestión de usuarios y funcionalidades avanzadas como:
- **Caching**: Optimización de consultas de juegos populares mediante Redis.
- **Validación de Esquemas**: Implementación de JSON Schema para reseñas externas.
- **Manejo de Archivos**: Sistema de carga de portadas de juegos.
